> Abstract. We give an explicit construction of two 2-generated subgroups H, K ≤SL(3 ,Z) whose inter-section is not finitely generated. The construction takes place inside the standard parabolic subgroup
> Z2⋊SL(2 ,Z)≤SL(3 ,Z). The main point is to identify H∩Kwith the stabilizer of a point for an affine action of a free group on Z2, and then to prove, using the Schreier graph of this action, that this stabilizer is not finitely generated. Furthermore, we prove that there exists a sequence of subgroups Hq, K q≤SL(3 ,Z)such that rank( Hq) = rank( Kq) = 4, and rank( Hq∩Kq)≥q+ 1 ,
> while Hq∩Kqis finitely generated.

1. Introduction 

Let G be a group. The rank of G, denoted rank( G), is the minimal cardinality of a generating set for G.In 1954, Howson [6] showed that the intersection of finitely generated subgroups of a free group is also finitely generated, which led to the notion of Howson group: a group G is said to be Howson , if the intersection 

H ∩ K of any two finitely generated subgroups H, K < G is again finitely generated. Many types of groups are Howson, for instance, free groups, surface groups, Baumslag-Solitar groups BS 1,n = ⟨a, t | tat −1 =

an⟩(n ≥ 1) [12], limit groups [3], etc. In particular, if G is a free or surface group, then rank( H ∩ K) − 1 ≤ (rank( H) − 1)(rank( K) − 1) ,

which was conjectured by Hanna Neumann in 1957, and proved independently by Friedman [4] and by Mineyev [11] for free groups in 2011, and by Antol´ ın and Jaikin-Zapirain [1] for surface groups in 2022. Note that the direct product Fn × Z of a free group Fn with rank n > 1 is not Howson. Moreover, Kapovich [7] showed many hyperbolic groups are not Howson. It is easy to show that the class of Howson groups is closed under taking subgroups and under finite extensions. In particular, virtually free groups and virtually surface groups are both Howson. More generally, the class of Howson groups is closed under graphs of groups, where the edge groups are finite [16]. Moreover, Shusterman and Zalesskii [15] extended the Howson property to the Demushkin group: the intersection of a pair of closed topologically finitely generated subgroups of a Demushkin group is topologically finitely generated. This gives the first example of the Howson pro-p groups that are not free. It is known that the special linear group SL(2 , Z) is Howson because it is virtually free. In contrast, SL( n, Z)( n ≥ 4) is not Howson [9], since it contains F2 × Z. Nevertheless, it still remains unclear whether SL(3 , Z) is Howson (see [9] and [8]). In this note we prove that SL(3 , Z) is not Howson: the intersection of two 2-generated subgroups can fail to be finitely generated. 

Theorem 1.1. The group SL(3 , Z) is not Howson.

2. Schreier graphs: basic facts 

We shall use only standard facts about Schreier graphs of subgroups of free groups. See Stallings [17] for the core-graph viewpoint and Lyndon–Schupp [10, Chapter I] for classical combinatorial group theory background. Let F = F (X) be the free group on a finite set X, and let N ≤ F be a subgroup. 

Definition 2.1 (Schreier graph) . The (right) Schreier graph Γ( N \F, X ) has vertex set N \F , the set of right cosets. For every vertex N g and every generator x ∈ X, there is a directed edge N g x

−→ N gx . Equivalently, one may include inverse edges labeled x−1.The following properties are the ones used in the proof of our main theorems. 

S1. Connectedness. The Schreier graph Γ( N \F, X ) is connected. Indeed, every vertex N g is reached from the base vertex N by reading a word representing g.

S2. Words and paths. A word in the alphabet X±1 labels a path in the Schreier graph. If the word is freely reduced, then the corresponding path has no immediate backtracking. 

S3. Loops and the subgroup. A word w ∈ F labels a loop at the base vertex N if and only if w ∈ N .More generally, w labels a loop at the vertex N g if and only if gwg −1 ∈ N , with the convention appropriate to right cosets. 

S4. Orbital interpretation. If F acts transitively on a set Ω and ω0 ∈ Ω, then the Schreier graph of Stab F (ω0) is naturally isomorphic to the orbital graph of the action on ω0F (The orbital graph has vertex set Ω = ω0F , and two vertices ω0g, ω 0gx are connected by an edge for x ∈ X). Under this isomorphism, the base vertex corresponds to ω0.

S5. The core. The core of a Schreier graph is the subgraph spanned by all edges and vertices lying on reduced closed paths. Equivalently, it is obtained by pruning off all hanging trees. A vertex that lies on a nontrivial reduced closed path lies in the core. Some form of the following is known to Stallings [17]. 

Lemma 2.2 (Stallings core criterion) . For a subgroup N ≤ F (X), the subgroup N is finitely generated if and only if the core of Γ( N \F, X ) is finite. In particular, if the core contains infinitely many vertices, then 

N is not finitely generated. Proof. A word w ∈ F labels a loop at the base vertex N in the graph Γ( N \F, X ) if and only if w ∈ N .Therefore, the subgroup N is the set P of labels of (reduced) closed paths based at the vertex N . Note that 

P can be viewed as a group (isomorphic to the group N ) with the product defined as the concatenation of labels of paths. Therefore, N is finitely generated if and only if the core is finite. □

Remark 2.3 . The Schreier graph is also referred to as the relative Cayley graph in some literature, for example, see [19, Section 2]. Given a left action of a group G on a set X, a right action can be defined via x · g := g−1 · x. Note that the stabilizers of a point in X are isomorphic for these two actions. While Schreier graphs and Cayley graphs usually adopt right multiplication, matrix actions adopt left multiplication. Following standard conventions, we adopt left group actions on sets throughout the subsequent two sections. The proofs of Theorems 1.1 and 1.2 rely on analyzing the corresponding stabilizers. 3.
